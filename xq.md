我们是一家钢化玻璃工厂，需要一款手机/平板可用的订单管理小程序或 App。

主要功能包括：

- 客户信息管理：有客户汇总，根据公司名字和日期分类。有在进行的订单的客户有不同颜色显示
- 订单创建（上传图纸 PDF）
- 订单状态流转（接单→生产→完成→取货）
- 工人更新生产状态
- 前台实时查看订单是否完成
- Pickup 时电子签字，替代纸质 pick up slip
- 不同角色权限（前台 / 工人 / 主管）
  
一、这个 App / 小程序的核心目标  
建立一套钢化玻璃厂订单全流程管理系统，实现从接单 → 生产 → 完工 → 取货签字 → 记录留存 的无纸化、实时可查流程，让前台、工人、管理者在手机或平板上各司其职。  
  
二、使用角色  
  
建议至少 4 类账号权限：  
  
1️⃣ 前台 / 接单人员（Office）  
        •        创建订单  
        •        查看订单状态  
        •        安排/通知生产  
        •        取货时调出订单  
  
2️⃣ 生产工人（Shop Floor）  
        •  有汇总可查看：分为公司和日期方式查看。点进去有在进行中的单子显示绿色，没有进行的就显示灰色。每个环节（切口，钢化环节）都要有一个选择哪一片玻璃要重做的按钮，要重做的玻璃都要自动发送至第一环节的切玻璃人员处，显示高亮，已读后高亮消失。  
      

•    可查看分配给自己的订单  
        •        更新生产状态  
        •        标记完成  
  
3️⃣ 管理者 / 主管（Supervisor）  
        •        审核订单  
        •        批准 pickup  
        •        查看所有订单 & 统计  
  
4️⃣ 顾客（不一定需要账号）  
        •        仅在取货时 现场签字  
        •        不需要登录（避免复杂）  
  
三、App / 小程序核心模块  
  
🔹 模块 1：用户登录 & 权限管理  
        •        手机号 / 邮箱登录  
        •        不同角色看到不同界面  
        •        简单权限控制（能看什么、能改什么）

•                “WhatsApp登录/微信号登录“  
  
🔹 模块 2：客户信息管理（Customer）  
  
用于重复客户，不用每次重填  
  
字段建议：  
        •        Company Name  
        •        Customer Name（可选）  
        •        Phone Number  
        •        Email（可选）  
        •        Notes（特殊要求）  
  
👉 可在下单时 一键选老客户  
  
🔹 模块 3：订单管理（Order System）⭐核心  
  
下单时录入内容：  
        •        Order Number（系统自动生成）有加急提示  
        •        Customer（选择 / 新建）  
        •        Glass Type（“默认Clear”）还有RAIN,PINHEAD,GREY,FROSTED…可选，可添加玻璃类型

        •        Thickness（如4mm/5mm/6mm / 10mm/ 12mm）  
        •        Quantity  
        •        Drawing / PDF 上传（图纸）  
        •        Special Instructions（“可选某片玻璃HOLD ”）  
        •        Priority（订单标签：Normal / Rework/Rush高亮/可选某片玻璃HOLD）  
        •        Estimated Completion Date  
        •                “  显示录入时间 ”             
               •                  新订单可以撤回订单，订单需要修改的时候有高亮提示

   •    前台输入订单后，按照玻璃厚度，数量和类型（有CLEAR,RAIN,PINHEAD,GREY,FROSTED…\)填入订单。 生产工人见到的订单按照厚度，类型和日期分类好。

超过5天时间没有完成工序的订单有高亮提示。

## <a id="_w5m5wjapvlt3"></a>__订单修改     允许：__

- 修改订单
- 修改图纸
- 修改数量
系统 __自动高亮提醒生产人员__

  
📌 关键点：  
图纸只需上传 PDF / 图片，不要求 App 画图  
（画图仍在你们现有电脑系统完成）  
  
🔹 模块 4：订单状态流转（Status Flow）⭐关键痛点解决  
  
建议固定状态：  
        1\.        🟡 Received（已接单）  
        2\.        🔵 Drawing / Entered in System（已画图/录入）  
        3\.        🟠 In Production（生产中）  
        4\.        🟢 Completed（已完成）  
        5\.        ✅ Ready for Pickup（可取货）

                   可以发送邮件通知客户拿货（可以更改邮件模板）  
        6\.        📦 Picked Up（已取）  
  
👉 工人或主管只需 点一下状态  
👉 前台实时看到，不再去车间问  
  
🔹 模块 5：生产工人界面（极简）  
  
工人只看到：  
        •        自己被分配的订单，只有上个阶段的工序完成了，才能发送到下一个工序的工人那里 （切玻璃工人 \- 开切口工人 \- 钢化工人 \- 完成钢化处工人）其中钢化工人只需要可以查看订单， 不需要多余操作  
        •        订单号 \+ 客户名，玻璃类型，数量，厚度  
        •        当前状态  
        •        【开始生产】  
        •        【生产完成】  
  
⚠ 不要复杂，越简单工人越愿意用  
  
🔹 模块 6：Pickup & 电子签字 ⭐你们最需要的  
  
取货流程：  
        1\.        前台搜索订单（订单号 / 客户名 / 手机/邮箱）  
        2\.        确认状态 = Ready for Pickup  
        3\.        批准pick up后才可以调出签名提货页面，否则不可签名提货。仓库工作人员收到通知，可协助顾客取货。

•                 •                        4\.        顾客在手机 / 平板上：  
        •         手写签名（电子签）  
        •         自动记录时间、取货人  
        5\.        状态自动变为 Picked Up  
                6\.                显示产品完成后几天未取的天数  
📌 系统自动保存：  
        •        签名图片  
        •        日期 & 时间  
        •        操作人  
📌 可导出 PDF（相当于 Pickup Slip）  
  
🔹 模块 7：历史记录 & 搜索  
        •        按客户查  
        •        按订单号查  
        •        按日期查  
        •        永久保存（至少几年）  
  
🔹 模块 8：通知  
        •        订单完成 → 前台收到提醒  
        •        可取货 → 前台提醒，然后仓库工人收到通知提醒。  
        •        Rush 订单高亮显示  
  
（不一定要短信，App 内提醒即可）  
  
四、设备 & 技术建议  
        •        📱 支持 手机 \+ iPad / 平板  
        •        🌐 Web \+ App 或 Web App（PWA）即可  
        •        ☁ 云端数据库（数据不丢）  
        •        🔐 数据权限分级  
        •        📄 支持 PDF 导出（订单 / Pickup 记录）

                 可英文，中文切换

  
  
👉 不建议一开始做太复杂的原生 App  
👉 小程序 / Web App 更省钱、上线快

        •        登录  
        •        客户管理  
        •        订单管理  
        •        状态更新  
        •        电子签字 \+ Pickup  
  
Phase 2（以后加）  
        •        报表（每天做多少）
